Feature: Amazon Shopping Cart Functionality

Scenario: Search, Add to Cart, and Update Quantity

Given the user is on the Amazon homepage
When the user searches for a product "Samsung Galaxy s24"  
Then the search results page is displayed 
When the user clicks on the "Add to Cart" button on the product card
Then a confirmation message is displayed indicating the product was added to the cart
When the user navigates to the shopping cart
Then the cart should display the added product with quantity "1"
When the user opens the quantity dropdown for the first item
And the user selects "2" as the new quantity
Then the quantity in the cart should be updated to "2"
